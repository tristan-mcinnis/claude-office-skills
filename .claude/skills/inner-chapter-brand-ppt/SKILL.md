## Inner Chapter Brand Guidelines

### Color Palette

**Primary**
- Black: `#000000` (primary text, dark backgrounds)
- White: `#FFFFFF` (light backgrounds, reversed text)

**Secondary**
- Dark Gray: `#262626` (secondary text, subtle backgrounds)
- Medium Gray: `#999999` (captions, tertiary text)

**Accent**
- Acme Blue: `#004479` (social branding, occasional accent - use sparingly)

### Typography

#### Font Stack

| Purpose | Font | Weight | Fallback |
|---------|------|--------|----------|
| Headlines | Helvetica Neue | Medium (500) | Arial |
| Body | Helvetica Neue | Light (300) | Arial |
| Data/Technical | Noto Mono | Regular | Courier New |

#### Size Guidelines (Presentations)

- Slide titles: 32-40pt
- Section headers: 24-28pt
- Body text: 14-18pt
- Captions/footnotes: 10-12pt
- Data tables: 10-14pt (Noto Mono)

### Logo Usage

**Primary Mark**
- Black outline version on white/light backgrounds
- White outline version on black/dark/colored backgrounds

**Placement (Presentations)**
- Title/cover slides: Stamp mark, centered or offset
- Content slides: "INNER CHAPTER" wordmark, top-left, small (10-12pt)
- Contact/closing slides: Full logo or wordmark as appropriate

**Clear Space**
- Minimum 0.5 inches around logo on all sides

### Slide Layout

**Margins**
- Minimum 0.5 inches (36pt) from all edges
- Breathing room is core to the aesthetic

**Grid**
- Prefer 2-column layouts for content-heavy slides
- Full-bleed photography permitted (content overlay with sufficient contrast)

**Slide Types**

| Type | Description |
|------|-------------|
| Title/Cover | Dark background, stamp logo, minimal text |
| Section Divider | Photography background or solid black, headline only |
| Content | Headline + body text or bullets |
| Image-Led | Full-bleed photo with caption/overlay |
| Client Logos | Grid arrangement (2x4 or 3x3) |
| Case Study | Headline + narrative + supporting image |
| Data/Table | Headline + table (Noto Mono for data) |
| Quote | Large quote text, attribution below |
| Team/Bio | Photos + text blocks |
| Contact | Contact details, logo, minimal design |

### Editorial Style

**Voice**
- Active voice always
- Clear, expressive, contextual, cultural
- American English default (British for UK clients)

**Formatting**
- Bullets: En-dash (â€“) not circles or diamonds
- Full stops: Omit at end of bullets and single sentences
- Numbers: Write numerically (3 not three)
- Quotes: "Double quotes" not 'single'
- Acronyms: Spell out first use, then abbreviate
- Caps: OK for headers and sub-headers

**Avoid**
- Italics for sub-headers (use bold)
- Underlines for emphasis
- Slang or undefined jargon
- Orphan words at line ends

### Icons

- Source from Noun Project (pro account) or Google Material Design
- Use sparingly
- Maintain consistency within a single deck (same icon set)
- Black or white only (no colored icons)

### Photography

- High-quality, editorial feel
- Prefer candid/documentary over staged
- Black and white treatment acceptable
- Full-bleed or contained in defined frames

### Client Adaptations

When creating materials for specific clients (Nike, Cartier, VF Corp, etc.):

**Keep consistent:**
- Typography (Helvetica Neue family)
- Layout grid and margins
- Editorial formatting rules
- Inner Chapter logo placement

**May adapt:**
- Add client brand colors as accents
- Include client logos per their guidelines
- Adjust tone for audience (more formal for luxury, more dynamic for sportswear)
